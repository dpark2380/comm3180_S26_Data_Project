# Project Plan

## Overview

This project investigates which global cities offer the highest quality of urban life relative to cost of living. We build an original "value for living" index by combining Numbeo city-level sub-indices (Healthcare, Crime, Pollution, Traffic) into a composite quality score, then dividing by Numbeo's Cost of Living Index. This keeps quality and cost analytically independent, unlike existing indices such as the EIU Global Liveability Index which ignore affordability entirely.

The project uses three data sources: Numbeo city-level indices (2019, 2020, 2021, 2022, 2024), World Bank World Development Indicators (country-level economic and health data), and WHO Global Health Observatory (country-level life expectancy and HALE).

The workflow moves from data collection and cleaning, through index construction and analysis, to a final data story and presentation.
